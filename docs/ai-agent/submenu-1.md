## 第1章 X3576核心板简介

### 1.1 产品简介
X3576是基于瑞芯微科技RK3576 CPU的一款核心板，由深圳市九鼎创展科技有限公司自主研发、生产并销售。  
RK3576是瑞芯微第二代8nm高性能AIOT平台，集成了独立的6TOPS NPU（神经网络处理单元），用于处理人工智能相关的任务。

### 1.2 核心板特性
- 最佳尺寸，保证引出全部GPIO口的同时，尺寸仅55mm*55mm；
- 系统供电使用PMU，保证工作稳定可靠，且成本低廉；
- 支持多种品牌、多种容量的emmc；
- 支持电源休眠唤醒；
- 支持千兆以太网、MIPI-CSI、MIPI-DSI、PCIE、USB3.0等高速总线；
- 采用208PIN邮票孔封装。

### 1.2.1 特性参数
- **CPU**：RK3576 (Quad A72 + Quad A53)
- **主频**：1.8GHz
- **RAM**：2GB 或 4GB 或 8GB
- **ROM**：4GB, 8GB, 16GB, 32GB 或 64GB

#### 接口参数
- **LCD接口**：1路 MIPI-DSI (MAX 2K @ 60Hz)
- **USBHOST 2.0接口**：2路
- **USBHOST 3.0接口**：2路
- **UART接口**：11路
